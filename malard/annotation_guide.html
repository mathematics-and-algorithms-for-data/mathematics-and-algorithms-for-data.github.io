The anatomy of the hand

This page contains instructions on how to annotate hand x-rays to generate training data for machine learning models. We use the ASPAX annotator software, which can be downloaded from here.

Bones of the fingers

Each finger (except the thumb) is composed of the following bones (by increasing distance from the wrist) . The thumb only has a distal phalanx (DP) and a proximal phalanx(PP). Every finger has a metacarpal bone (MC) which connects it to the wrist.

  1. and the metacarpal (MC)
  2. the proximal phalanx(PP)
  3. the middle phalanx (MP)
  4. and the distal phalanx (DP)

The thumb has the following bones (by increasing distance from the wrist)

  1. the metacarpal (MC)
  2. the proximal phalanx(PP)
  3. and the distal phalanx (DP)

Joints of the fingers

Each finger (except the thumb) has the following joints (by increasing distance from the wrist)

  1. the carpometacarpal joint (CMC)-between the wrist and the metacarpal bone
  2. the metacarpophalangeal joint (MCP)-between the metacarpal and the proximal phalanx
  3. the proximal interphalangeal joint (PIP)-between the proximal phalanx and the middle phalanx, and the distal
  4. interphalangeal joint (DIP)-between the middle phalanx and the distal phalanx.

The thumb has the following joints (by increasing distance from the wrist):

  1. Carpometacarpal Joint (CMC)-between the metacarpal and the wrist
  2. the metacarpophalangeal joint (MCP)-between the metacarpal and the proximal phalanx
  3. and the interphalangeal joint (IP)-between the distal phalanx and the proximal phalanx.

Bones of the Wrist

The wrist is made up of the following bones:

  1. the hamate,
  2. the capitate,
  3. the pisiform,
  4. the triquetrum,
  5. the trapezium,
  6. the trapezium,
  7. and the scaphoid.

Annotation instructions

ASPAX allows the user to draw polylines or rectangles around areas of interest. The polylines are used to delineate the bones of the hand, while the rectangle is used to highlight joint locations

More information on how to use ASPAX for annotation can be found here. Below is a summary of the annotation work scheme.

Annotating a bone

To annotate a bone, click on bone in the left hand menu, select the which hand you are annotating by clicking on 'L' or 'R'. Then start typing the abbreviation for the bone that you want to annotate. The dropdown should print out suggestions.

Next, make sure that you select the polyline option in the top tool pane. The icon should look like two dots joined by a line.

Make sure to start at 9 0'clock on the bone contour and that the point cloud is dense enough. A good way to know if the point cloud is dense enough is to check if the line between two dots is crossing the boundary of the bone. This line should be tangential to the bone contour. To annotate the next bone, clear the current annotations by clicking on the brush icon in the tool pane and repeat the above steps.

Annotating a Joint

To annotate a bone, click on 'Joint' in the left hand menu, select the which hand you are annotating by clicking on 'L' or 'R'. Then start typing the abbreviation for the joint that you want to annotate. The dropdown should print out suggestions.

Next, make sure that you select the rectangle option in the top tool pane. The icon should look like a rectangle. When drawing the rectangle, make sure that it is the smallest rectangle that contains the gap between the respective bones. To annotate the next joint, clear the current annotations by clicking on the brush icon in the tool pane and repeat the above steps.
